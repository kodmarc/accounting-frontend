# Frontend Module Documentation

This folder explains the frontend modules of the accounting software in a simple, feature-wise way.

## Main sections

- [Home](home.md) — dashboard summary, quick actions, and business health overview.
- [Authentication](authentication.md) — login, OTP verification, invitation acceptance, and logout flow.
- [Sales](sales/README.md) — invoices, quotes, projects, and sales follow-up.
- [Purchases](purchases/README.md) — bills, purchase orders, supplier actions, and payments.
- [Accounting](accounting/README.md) — chart of accounts, journals, bank transactions, tax setup.
- [Reports](reports/README.md) — profit and loss, balance sheet, cash flow, and account transactions.
- [Inventory](inventory.md) — products, stock, quantity-on-hand, and movement history.
- [Contacts](contacts.md) — customers, suppliers, and contact profile management.
- [Organization](organization.md) — members, roles, permissions, and company team operations.
- [Settings](settings.md) — company configuration, documents, bank setup, and account preferences.
- [Profile](profile.md) — personal user profile and account-owned details.

## How to read these docs

Each page explains:
- the purpose of the module
- the main user actions
- the important inputs required from the user
- the output or result produced after the action
- the related screens and workflows
